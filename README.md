Приложение БИБЛИОТЕКА.
=====================================
 Стек технологий:
------------------
* Spring-boot
* Hibernate

 Для сборки проекта и хоста сервера:
--------------------------
* Maven 
* Tomcat
## Функционал:
* возврат всех книг
* возврат всех жанров(без ссылки на все книги)
* возврат всех авторов(без ссылки на все книги)
* удаление книги с определенным id
* добавление книги
* изменение книги
* получение книги по id
